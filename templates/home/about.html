Technology

The LibCrowds platform aims to increase access to the collections of the British Library through a range of experimental crowdsourcing projects. The platform was first launched in June 2015 with the Convert-a-Card project series, which used a new method for transforming printed card catalogues into electronic records.

Each LibCrowds project series uses a seperate instance of PyBossa, which is an open-source framework for the creation of crowdsourcing projects. A custom theme provides targeted information for each project series and a range of Python plugins have been developed to provide additional functionality, such performing database searches via the Z39.50 protocol.

The LibCrowds theme, all project templates and plugins are available on GitHub.
