The {{ brand }} Platform

{{ brand }} is platform that aims to increase access to the collections of the British Library through a range of experimental crowdsourcing projects. The platform was first launched in June 2015 with the Convert-a-Card project series, which used a new method for transforming printed card catalogues into electronic records.

When the Playbills project series was launched two years later, in June 2017, a decision was made to define each project series using a separate instance of PyBossa, which is an open-source framework for the creation of crowdsourcing projects. A new theme was written that can be modified to provide a consistent look and feel, and targeted information for each project series. Alongside the theme a range of plugins have been developed to provide additional functionality {% if 'pybossa_z3950' in plugins %} , such as the plugins['pybossa_z3950'].name that allows for catalogue records to be retreived via Z39.50 searches {% endif %}.
