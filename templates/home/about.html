The {{ brand }} Platform

{{ brand }} is platform that aims to increase access to the collections of the British Library through a range of experimental crowdsourcing projects. The platform was first launched in June 2016 with the Convert-a-Card project series, which used a new method for transforming printed card catalogues into electronic records.

{{ brand }} is built on top of PyBossa , an open-source framework for the creation of crowdsourcing projects. {% if 'pybossa_discourse' in plugins %} The community forum is powered by Discourse.{% endif %} Each project on the platform is written in a combination of HTML, CSS and Javascript. A range of plugins have also been developed to provide additional functionality.
